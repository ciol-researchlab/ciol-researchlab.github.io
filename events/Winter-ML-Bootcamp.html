CIOL Presents Winter ML Bootcamp

Overview

CIOL Winter ML Bootcamp is an advanced technical workshop designed for CIOL members and external participants. This month long bootcamp focuses on practical machine learning (ML) applications, assignments, and research-oriented projects. Participants will gain hands-on experience, contribute to publishable research, and build a strong GitHub portfolio.

Register here: CIOL Winter ML Bootcamp | Registration

Event Details

  • Event Name: CIOL Winter ML Bootcamp
  • Prerequisite: Basic knowledge of Python, including input, variable handling, lists, dictionaries, functions, and file operations.
  • Schedule:
    • Date26 December, 2024 to 06 February, 2025
    • Start Time: 8:00–9:00 PM
    • Duration per Class: 2–2.5 hours
    • Gap Between Classes: 2 days (for assignment submission)
    • Total Class Time: ~16–20 hours
    • Assignment Time: ~20–30 hours
  • Platform: Zoom
  • Participants:
    • Mandatory for CIOL members
    • External participants can register (limited seats available)
    • Recorded sessions available (no certificates or labs)

Session Plan

  1. Exploratory Data Analysis (EDA): Data cleaning and visualization (Assignment: Perform EDA on a dataset)
  2. Tabular Data Model Training: Building and evaluating models (Assignment: Train and evaluate a model)
  3. Hyperparameter Tuning: Optimize and select the best model (Assignment: Tune models)
  4. Pre-trained Models for Text: NLP tasks using BERT and variants (Assignment: Text classification)
  5. Pre-trained Models for Images: Using CNNs and ViT (Assignment: Image classification)
  6. LLM Fine-tuning: Fine-tune large language models (Assignment: Fine-tune a small-scale model)
  7. Explainable AI (XAI): Interpreting ML models (Assignment: Generate explanations for predictions)
  8. Research (Problem to Code): From research problems to actionable code (Final Assignment: Research proposal with analysis)
